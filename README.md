## 🧠 LLM Prompt Library — Level: Advanced

> *Prompt yang tepat = karyawan virtual yang tidak pernah sakit dan tidak minta naik gaji.*

### 💸 Money-Making Prompts (Battle-Tested)

**Client Acquisition — Cold DM Template**
```
Kamu adalah copywriter senior dengan pengalaman 15 tahun.
Tulis cold DM untuk [platform] kepada [target client] di bidang [industri].
Tujuan: booking discovery call dalam 48 jam.
Tone: percaya diri, tidak sombong, problem-aware.
Panjang: max 3 paragraf. Sertakan 1 social proof subtle.
Output: 3 variasi dengan angle berbeda.
```

**SEO Article Generator — Rank-Optimized**
```
Kamu adalah SEO content strategist.
Tulis artikel [X kata] tentang [topik] untuk target keyword "[keyword]".
Struktur: H1 → intro (PAS framework) → [5-7 H2 subheading] → FAQ → CTA.
Masukkan keyword density 1-1.5%, LSI keywords natural.
Tone: [formal/casual], target audience: [deskripsi].
Output: artikel siap publish + meta description 155 karakter.
```

**YouTube Script — High Retention Format**
```
Buat script YouTube untuk video berdurasi [X menit] tentang [topik].
Format: Hook 15 detik (pattern interrupt) → Promise → Content → CTA.
Masukkan [jumlah] open loop untuk jaga retention.
Tone: [energetic/calm/educational].
Target: [deskripsi audience].
Output: script lengkap + timestamp.
```

**Product Description — E-commerce Converter**
```
Tulis deskripsi produk untuk [nama produk] yang dijual di [platform].
Features: [list fitur].
Target buyer: [deskripsi].
Gunakan framework: Feature → Benefit → Emotional Outcome.
Panjang: 150-200 kata. Sertakan 5 bullet points.
CTA di akhir: urgensi tanpa manipulasi.
```

**Trading Analysis Prompt — AI Signal Filter**
```
Analisa chart [pair] timeframe [TF] berdasarkan data berikut:
Harga saat ini: [X]
Support/Resistance: [level]
Indikator: [RSI, MACD, dll]

Identifikasi: setup entry terbaik, level SL, TP1, TP2.
Risk/Reward minimum: 1:2.
Berikan confidence score (1-10) dan alasan.
Output: entry brief dalam format tabel.
```

## 🛡️ Cost & Security Notes

> *Jangan sampai semangat cuan malah jadi buntung karena tidak hati-hati.*

### 💸 Cost Considerations

- **API Costs:** Selalu set billing alert dan hard limit di dashboard provider AI kamu
- **LLM Pricing:** GPT-4o, Claude Sonnet, Gemini 1.5 Pro — bandingkan per token sebelum scale
- **Automation Hosting:** VPS $4–$6/bulan cukup untuk 90% use case di repo ini
- **Free Alternatives:** Groq (free, fast), Google AI Studio (free tier generous), Mistral API

### ⚠️ Security Warnings

- **JANGAN** simpan API key di public repo. Pakai `.env` dan `.gitignore`
- **JANGAN** jalankan trading bot dengan modal besar sebelum backtest minimal 3 bulan
- **JANGAN** share private key / seed phrase ke tool AI apapun
- Selalu gunakan API key dengan permission minimal (read-only kalau cukup)
- Paper trade dulu, real money belakangan
